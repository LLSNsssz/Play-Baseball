## 🎯 프로젝트 소개

> "어제의 물건이 내일의 보물로"

**SecondInning**은 AI 기반 중고거래 플랫폼입니다. 우리는 사용하지 않는 물건에 새로운 생명을 불어넣고, 지속 가능한 소비 문화를 만들어갑니다.

### 💡 우리의 미션

1. 🌱 지속 가능한 소비 문화 조성
2. 🤝 신뢰할 수 있는 거래 환경 구축
3. 🔍 AI 기반 상품 출고가 표시
4. 💰 합리적인 가격 책정 지원

## 🚀 시작하기

1️⃣ **저장소 복제**
```bash
git clone https://github.com/Ormi5-StrikeOut/Play-Baseball.git
cd Play-Baseball
```

2️⃣ **의존성 설치**
```bash
# 프론트엔드
npm install

# 백엔드
./gradlew build
```

3️⃣ **환경 설정**
```bash
cp .env.example .env
# .env 파일에 필요한 환경 변수를 설정하세요
```

4️⃣ **실행**
```bash
# 프론트엔드
npm run dev

# 백엔드
./gradlew bootRun
```

🎉 `http://localhost:3000`에서 SecondInning을 만나보세요!
